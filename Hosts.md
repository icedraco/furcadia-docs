Furcadia Hosts & DNS
===============================================================================

Overview
-------------------------------------------------------------------------------
The following is a list of Furcadia hosts and their designated IP addresses.
This list is meant to provide an archive and a means to restore connectivity
to Furcadia resources in an event of local or global DNS failure.

The hosts in this list were publically sourced - either from the website, or
game software making DNS lookups. If you wish to contribute to this list,
please bear this in mind and avoid disclosing privileged information that you
have no authority to publish.
